Our Work

DNS Resolver — A set of C++ classes for DNS resolver. It is easy to use these classes in your applications to retrieve DNS resource records.

HTTP Parser — HTTP message parser used for HTTP server and client. It can be generalized to a text-based message protocol.

RTSP server — A simple RTSP server embedded in applications. RTSP session management and RTSP message parser.

RTP Packer — Build SDP and RTP packet from a video file. Support file formats of ASF/WMV and RM/RMVB at this moment.

STUN Client — STUN message parser and a stun client for NAT discovery. Compliant with IETF RFC 3489 spec.

PPIO — PPIO is a open-source projet to develop a C++ library for peer-to-peer communications.

Clip Sender — Capture contents in system clip board and send to a given URL, triggered by a keyboard shortcut.
