All Projects

BetterDrive is a improved personal cloud storage system compared to similar systems like Transporter, Lima, and SherlyBox, which let users own and control their data absolutely. BetterDrive leveraged existing technologies like peer-to-peer networking, WebRTC/ORTC, and VPN (virtual private network).
BetterCast is a improved video broadcasting system that developed on the base of PPEngine and PPCast. It provides on-demand and live video streaming over Internet, and works in a peer-to-peer fashion.
BetterEye is a smart video surveillance system that leveraged the power of OpenCV. It supports remote surveillance over Internet and provides smart features like silence and motion detection, face detection and recognition.
PPEngine is a software framework for peer-to-peer (P2P) networking.
PPCast is a Internet video broadcasting system.
FireWind is a Internet video broadcasting system.
GoldenEye is a networking video surveillance system.
