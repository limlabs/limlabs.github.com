File Splitter

This is a command line tool to split a large text file into multiple smaller files. Please find source code at https://github.com/limlabs/filesplitter.

The program was tested on Mac OS X Mavericks (10.9.2) and Windows XP (SP3). It works well with text file include millions lines.
